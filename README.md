# Real time chat app
Simple chat app written in Swift 5 using Firebase

## Features
- Facebook  Log In
- Google Sign In
- Email/Pass Registration / Log In
- Photo Messages
- Video Messages
- Real Time Conversations
- Location Messages
- Search for Users
- Deleting Conversations
- User Profile

## Set Up
```
git clone git@github.com:menescots/Messenger.git
```
